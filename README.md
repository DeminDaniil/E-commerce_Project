# E-commerce_Project
Проект по электронной коммерции - продажа одежды.  Была проведена очистка данных и их анализ с помощью Python, построение дашборда в Power BI
В данном проекте была проделана работа по очистке данных, так как было большое количество данных, которые
было бы неудобно анализировать, или они портили статистику

В качестве анализа данных на Python, было проведено следующее:
  - Чистка и корректировка данных
  - Подсчитано количество товаров в каждой категории одежды
  - Подсчитано количество товаров в каждой категории скидок
  - Подсчитано количество товаров в каждой категории размеров
  - Самые продаваемые бренды (global desi, w, new balance)

В Power BI в дашборде приведены следующие результаты:
  - Сумма стоимостей товаров:
    - Со скидкой
    - Без скидки
  - Средняя розничная цена до скидки и после
  - Сумма розничных цен по категориям одежды
  - Количество различных скидок в каждой категории одежды
  - Количество товаров с различным типом скидок
